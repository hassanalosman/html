Web Development Terminologies

Computer Science & Web Development Terminologies

HTML
HyperText Markup Language used to structure content on the web.
CSS
Cascading Style Sheets used to style and layout web pages.
JavaScript
A programming language that adds interactivity to websites.
Frontend Development
The part of web development that deals with user interfaces.
Backend Development
Server-side development handling logic, databases, and APIs.
Responsive Design
Design approach that makes websites work on all screen sizes.
Semantic HTML
HTML elements that clearly describe their meaning and role.
DOM
Document Object Model representing HTML elements as objects.
API
Application Programming Interface for communication between systems.
HTTP
Protocol used for transferring data over the web.
HTTPS
Secure version of HTTP using encryption.
URL
Uniform Resource Locator, the address of a web resource.
Web Browser
Software used to access and display web pages.
Web Server
System that stores and delivers web content to browsers.
Framework
A pre-built structure to speed up web development.
Library
Collection of reusable code for specific tasks.
Hosting
Service that makes a website available on the internet.
Version Control
System for tracking changes in code over time.
Git
A popular distributed version control system.
SEO
Search Engine Optimization to improve website visibility.
